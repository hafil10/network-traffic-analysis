# Network Traffic Analysis & DNS Investigation

## Project Overview

This project demonstrates hands-on network traffic analysis using Kali Linux and Wireshark.

The objective was to capture, inspect, and analyze network traffic generated during common network activities and investigate DNS queries and responses.

---

## Tools Used

* Kali Linux
* Wireshark
* Linux Terminal

---

## Activities Performed

### ICMP Traffic Analysis

* Captured live network traffic using Wireshark
* Generated ICMP traffic using ping commands
* Filtered packets using Wireshark display filters
* Analyzed Echo Request and Echo Reply packets
* Saved packet capture evidence

### DNS Investigation

* Captured DNS queries and responses
* Investigated domain name resolution
* Examined DNS packet details
* Identified queried domains and returned IP addresses
* Documented findings and evidence
